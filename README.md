 # Projeto-Rede Social 
![readme](https://github.com/user-attachments/assets/43a636ba-c4e5-4f74-a627-5c0034ecfb01)

A Conexo é uma plataforma social inovadora, criada com o objetivo de integrar pessoas em um ambiente digital moderno, funcional e acessível. Mais do que uma rede social comum, a Conexo se destaca como um espaço pensado especialmente para gamers, oferecendo funcionalidades únicas para que entusiastas de jogos possam registrar seu progresso, compartilhar conquistas, comentar seus favoritos, postar reviews e reviver momentos épicos.

Este projeto foi desenvolvido como trabalho de conclusão da disciplina de Desenvolvimento Web.

#### Funcionalidades Principais
 A plataforma Conexo oferece uma ampla variedade de recursos voltados à experiência do usuário:

#### Feed Personalizado: 
Visualize postagens com textos, imagens, vídeos e músicas, com conteúdo exibido de acordo com as preferências e interações de cada usuário.

Stories Temporários: Compartilhe conteúdos que desaparecem após um determinado tempo, promovendo publicações rápidas e espontâneas.

Sistema de Chat Completo: Comunique-se através de mensagens de texto, emojis, GIFs, arquivos, áudios e chamadas de voz ou vídeo.
![Chat](https://github.com/user-attachments/assets/e95ccac6-6efd-4490-a565-99d593645c25)


#### Perfil do Usuário: 
Personalize seu perfil com foto de capa, avatar, biografia, localização e destaques visuais para expressar sua identidade

Desfrute de um reprodutor de áudio acessível em todas as páginas, permitindo escutar músicas enquanto navega pela plataforma.

## Tecnologias Utilizadas
  O projeto Conexo foi desenvolvido com foco em tecnologias de front-end, garantindo uma interface responsiva, dinâmica e intuitiva:

HTML5: Responsável pela estrutura semântica e básica das páginas.

CSS3: Aplicação de estilo visual com design responsivo, animações e uso de variáveis para personalização de temas (claro/escuro).

JavaScript (Vanilla JS): Lógica de interação, controle de elementos e funcionalidades dinâmicas em toda a plataforma.

Font Awesome: Conjunto de ícones escaláveis e vetoriais utilizados em toda a interface para uma experiência visual consistente.

Google Fonts (Inter): Fonte utilizada para uma leitura limpa, moderna e acessível.

Giphy API: Integração com a biblioteca Giphy para envio e exibição de GIFs no sistema de chat, enriquecendo a comunicação.

## Como Rodar o Projeto Localmente
Para configurar e executar o projeto Conexo em sua máquina local, siga os passos abaixo:

## Clone o Repositório:

git clone https://github.com/elberthmayan/conexo.git

Navegue até o Diretório do Projeto:

cd conexo

Abra os Arquivos no Navegador:
Como este é um projeto puramente de front-end, você pode simplesmente abrir os arquivos .html diretamente no seu navegador. Comece com index.html ou feed.html.

Recomendado: Para um melhor ambiente de desenvolvimento (especialmente para APIs que exigem HTTPS ou para evitar problemas de CORS), você pode usar uma extensão de servidor local para VS Code (como "Live Server") ou um servidor HTTP simples do Python:

# No diretório do projeto
python -m http.server 8000

Em seguida, acesse http://localhost:8000 no seu navegador.

Acessando a Plataforma (Login):
Ao abrir o projeto, você será redirecionado para a página de login. Para acessar o feed e as funcionalidades da plataforma, utilize as seguintes credenciais de teste:

Email: Qualquer valor que contenha @gmail.com (ex: teste@gmail.com, usuario123@gmail.com).

Senha: Qualquer valor.
